# SolarPlot

1. Make sure to clone repo into your Desktop to make sure "solar" script works
2. Open StellarBodyPlotting in IntelliJ or another Java IDE
3. Run the code to create the data files
4. Ensure the "solar" script has permissions to be run by entering
  * chmod +x solar
5. Finally run the "solar" script in the terminal by entering
  * ./solar
